Unit 1: HTML

In unit 1 we started to learn the basics of Hyper Text Markup Language, and what some of the elements used in the creation of a website means, such as tags. Another part of our learning was the internet, and it's history.

Unit 2: CSS

In unit 2, we learned and used CSS and how to incorperate it into the websites we are building. This included aligning, coloring, and add much more style then what could be done before.

Unit 3: Web graphics

Web graphics included learning about how to use images to support the attractiveness of your website without being overdone or annoying. Unit 3 also taught how to get and use images legally.

Unit 4: JavaScript

Javascript showed how to make websites more interacitve or have more complexity incorperated into the design of it. This included learning how to make java do complex math with multiple variables, or creating a pixelart webpage.
